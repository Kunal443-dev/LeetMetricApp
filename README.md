##LeetMetricApp:
A responsive web app to view LeetCode user statistics by entering a username.
Displays problem-solving progress and submission stats using LeetCode GraphQL API.

## Features:
Search LeetCode user by username
Easy / Medium / Hard progress visualization
Dynamic stats cards for submissions
Responsive and interactive UI

## Tech Stack:

HTML5
CSS3
JavaScript (ES6+)
GraphQL (LeetCode API)

##Project Structure:
LeetMetric/
├── index.html
├── style.css
├── leetmetric.js
    

## How to Run

Clone the repository:
git clone https://github.com/Kunal443-dev/LeetMetricApp.git
Open index.html in a browser
Enter a valid LeetCode username and click Search


#### Notes:

Frontend demo only; uses a public CORS proxy for API requests
For production, backend integration is recommended
